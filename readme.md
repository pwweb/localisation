## Installation

Via Composer run the following:

``` bash
# Install the package.
$ composer require pwweb/localisation

# Publish config and migration.
$ php artisan vendor:publish --provider="PWWeb\Localisation\LocalisationServiceProvider"

# Run migrations
$ php artisan migrate
```
